## Docker API Architecture

Every `docker` command you have ever run is just a REST API call in disguise. The Docker CLI is a client that sends HTTP requests to the Docker daemon.

```
Docker Architecture — The API Layer:
┌────────────────────────────────────────────────────────────────┐
│                                                                 │
│  ┌─────────────┐    HTTP/REST    ┌──────────────────────┐      │
│  │  Docker CLI  │───────────────▶│   Docker Daemon      │      │
│  │  (client)    │                │   (dockerd)          │      │
│  └─────────────┘                 │                      │      │
│                                  │  ┌────────────────┐  │      │
│  ┌─────────────┐    HTTP/REST    │  │  Container     │  │      │
│  │  curl        │───────────────▶│  │  Runtime       │  │      │
│  │  (raw API)   │                │  │  (containerd)  │  │      │
│  └─────────────┘                 │  └────────────────┘  │      │
│                                  │                      │      │
│  ┌─────────────┐    HTTP/REST    │  ┌────────────────┐  │      │
│  │  Python SDK  │───────────────▶│  │  Image Store   │  │      │
│  │  (docker-py) │                │  │                │  │      │
│  └─────────────┘                 │  └────────────────┘  │      │
│                                  │                      │      │
│  ┌─────────────┐    HTTP/REST    │  ┌────────────────┐  │      │
│  │  Custom App  │───────────────▶│  │  Network       │  │      │
│  │  (any lang)  │                │  │  Driver        │  │      │
│  └─────────────┘                 │  └────────────────┘  │      │
│         │                        └──────────────────────┘      │
│         │                                                       │
│         ▼                                                       │
│  Transport Options:                                             │
│  ├── Unix Socket: /var/run/docker.sock (default, local only)   │
│  ├── TCP: tcp://0.0.0.0:2376 (remote, must use TLS)           │
│  └── SSH: ssh://user@host (remote, via SSH tunnel)             │
│                                                                 │
└────────────────────────────────────────────────────────────────┘
```

### The Unix Socket

By default, Docker listens on a Unix socket at `/var/run/docker.sock`. This is a file on the filesystem that allows local processes to communicate with the Docker daemon.

```bash
# Check the socket exists
ls -la /var/run/docker.sock
# srw-rw---- 1 root docker 0 Mar 01 10:00 /var/run/docker.sock

# The socket is owned by root:docker
# Your user must be in the 'docker' group to access it
groups
# user docker

# The Docker CLI uses this socket by default
# You can see this with strace:
strace -e trace=connect docker ps 2>&1 | grep docker.sock
# connect(3, {sa_family=AF_UNIX, sun_path="/var/run/docker.sock"}, 33) = 0
```

<Callout type="warning" title="The Docker Socket Is Root-Equivalent">
Any process that can access `/var/run/docker.sock` has full control over the Docker daemon, which means it can mount the host filesystem, access host networking, and effectively gain root access to the host. Never mount the Docker socket into containers unless absolutely necessary, and never expose it over TCP without TLS and authentication.
</Callout>

## Interacting with the API via curl

You can talk to the Docker API using `curl` and the `--unix-socket` flag. This is the most direct way to understand what the API offers.

### Ping and Version

```bash
# Ping the daemon — simplest possible API call
curl --unix-socket /var/run/docker.sock http://localhost/_ping
# OK
```

### Container Operations

```bash
# List all running containers (equivalent to `docker ps`)
curl -s --unix-socket /var/run/docker.sock \
  "http://localhost/containers/json" | jq '.[] | {
    Id: .Id[:12],
    Name: .Names[0],
    Image: .Image,
    State: .State,
    Status: .Status
  }'
# {
#   "Id": "abc123def456",
#   "Name": "/auth-service",
#   "Image": "registry.acme.io/platform/auth-service:v2.3.1",
#   "State": "running",
#   "Status": "Up 3 hours (healthy)"
# }

# List ALL containers including stopped (equivalent to `docker ps -a`)
curl -s --unix-socket /var/run/docker.sock \
  "http://localhost/containers/json?all=true" | jq length
# 12

# Create a new container (equivalent to `docker create`)
curl -s --unix-socket /var/run/docker.sock \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "Image": "nginx:alpine",
    "Cmd": ["nginx", "-g", "daemon off;"],
    "ExposedPorts": { "80/tcp": {} },
    "HostConfig": {
      "PortBindings": {
        "80/tcp": [{ "HostPort": "8080" }]
      },
      "Memory": 268435456,
      "CpuQuota": 50000
    },
    "Env": ["NGINX_HOST=example.com"],
    "Labels": {
      "app": "web",
      "environment": "dev"
    }
  }' \
  "http://localhost/containers/create?name=my-nginx" | jq .
# {
#   "Id": "a1b2c3d4e5f6...",
#   "Warnings": []
# }

# Start the container
CONTAINER_ID="a1b2c3d4e5f6"
curl -s --unix-socket /var/run/docker.sock \
  -X POST \
  "http://localhost/containers/${CONTAINER_ID}/start"
# (empty response, 204 No Content = success)

# Inspect a container (equivalent to `docker inspect`)
curl -s --unix-socket /var/run/docker.sock \
  "http://localhost/containers/${CONTAINER_ID}/json" | jq '{
    State: .State,
    Config: { Image: .Config.Image, Env: .Config.Env },
    NetworkSettings: { IPAddress: .NetworkSettings.IPAddress }
  }'

# Get container logs (equivalent to `docker logs`)
curl -s --unix-socket /var/run/docker.sock \
  "http://localhost/containers/${CONTAINER_ID}/logs?stdout=true&stderr=true&tail=20"

# Get container stats (equivalent to `docker stats --no-stream`)
curl -s --unix-socket /var/run/docker.sock \
  "http://localhost/containers/${CONTAINER_ID}/stats?stream=false" | jq '{
    cpu_percent: ((.cpu_stats.cpu_usage.total_usage - .precpu_stats.cpu_usage.total_usage) / (.cpu_stats.system_cpu_usage - .precpu_stats.system_cpu_usage) * .cpu_stats.online_cpus * 100),
    memory_usage_mb: (.memory_stats.usage / 1048576),
    memory_limit_mb: (.memory_stats.limit / 1048576),
    network_rx_mb: ((.networks.eth0.rx_bytes // 0) / 1048576),
    network_tx_mb: ((.networks.eth0.tx_bytes // 0) / 1048576)
  }'

# Stop a container (equivalent to `docker stop`)
curl -s --unix-socket /var/run/docker.sock \
  -X POST \
  "http://localhost/containers/${CONTAINER_ID}/stop?t=10"
# (waits up to 10 seconds for graceful shutdown, then sends SIGKILL)

# Remove a container (equivalent to `docker rm`)
curl -s --unix-socket /var/run/docker.sock \
  -X DELETE \
  "http://localhost/containers/${CONTAINER_ID}?force=true"
```

### Image Operations

```bash
# List images (equivalent to `docker images`)
curl -s --unix-socket /var/run/docker.sock \
  "http://localhost/images/json" | jq '.[] | {
    Id: .Id[:19],
    RepoTags: .RepoTags,
    Size_MB: (.Size / 1048576 | floor)
  }'

# Pull an image (equivalent to `docker pull`)
curl -s --unix-socket /var/run/docker.sock \
  -X POST \
  "http://localhost/images/create?fromImage=alpine&tag=3.19" | jq .
# {"status":"Pulling from library/alpine","id":"3.19"}
# {"status":"Pulling fs layer","progressDetail":{},"id":"sha256:abc123..."}
# {"status":"Pull complete","progressDetail":{},"id":"sha256:abc123..."}
# {"status":"Digest: sha256:def456..."}
# {"status":"Downloaded newer image for alpine:3.19"}

# Pull from a private registry (with authentication)
# First, encode credentials
AUTH=$(echo -n '{"username":"myuser","password":"mypass","serveraddress":"registry.acme.io"}' | base64)

curl -s --unix-socket /var/run/docker.sock \
  -X POST \
  -H "X-Registry-Auth: ${AUTH}" \
  "http://localhost/images/create?fromImage=registry.acme.io/platform/auth-service&tag=v2.3.1"

# Tag an image (equivalent to `docker tag`)
curl -s --unix-socket /var/run/docker.sock \
  -X POST \
  "http://localhost/images/alpine:3.19/tag?repo=registry.acme.io/platform/base-alpine&tag=3.19"

# Delete an image (equivalent to `docker rmi`)
curl -s --unix-socket /var/run/docker.sock \
  -X DELETE \
  "http://localhost/images/alpine:3.18" | jq .
# [
#   { "Untagged": "alpine:3.18" },
#   { "Deleted": "sha256:abc123..." }
# ]

# Build an image from a Dockerfile (equivalent to `docker build`)
# The build context must be sent as a tar archive
tar -czf context.tar.gz Dockerfile src/ requirements.txt

curl -s --unix-socket /var/run/docker.sock \
  -X POST \
  -H "Content-Type: application/x-tar" \
  --data-binary @context.tar.gz \
  "http://localhost/build?t=myapp:latest&dockerfile=Dockerfile"
```

<Callout type="info" title="API Reference Documentation">
The complete Docker Engine API reference is available at https://docs.docker.com/engine/api/. Each endpoint is documented with request parameters, request body schema, and response format. The API version in the URL (e.g., `/v1.45/containers/json`) determines which features are available. Always specify the API version explicitly in production code to avoid breaking changes.
</Callout>

```python
# docker_basics.py — Basic container operations with docker-py
import docker

# Connect to the Docker daemon
client = docker.from_env()  # Uses /var/run/docker.sock by default

# Alternatively, connect to a remote Docker host
# client = docker.DockerClient(base_url='tcp://192.168.1.100:2376',
#                               tls=True)

# --- System Information ---
info = client.info()
print(f"Docker Version: {info['ServerVersion']}")
print(f"Containers: {info['Containers']} (Running: {info['ContainersRunning']})")
print(f"Images: {info['Images']}")
print(f"CPUs: {info['NCPU']}, Memory: {info['MemTotal'] // (1024**3)} GB")

# --- Container Operations ---

# List running containers
for container in client.containers.list():
    print(f"  {container.short_id} | {container.name:20s} | {container.status} | {container.image.tags}")

# Run a new container (docker run equivalent)
container = client.containers.run(
    image="nginx:alpine",
    name="api-test",
    ports={'80/tcp': 8080},
    environment={'NGINX_HOST': 'localhost'},
    labels={'app': 'test', 'managed-by': 'python-sdk'},
    mem_limit='256m',
    cpu_quota=50000,
    detach=True,  # Run in background
    remove=False
)
print(f"Started container: {container.name} ({container.short_id})")

# Get container details
container.reload()  # Refresh state from daemon
print(f"Status: {container.status}")
print(f"IP Address: {container.attrs['NetworkSettings']['IPAddress']}")

# Get container logs
logs = container.logs(tail=10, timestamps=True).decode('utf-8')
print(f"Last 10 log lines:\n{logs}")

# Execute a command inside the container
exit_code, output = container.exec_run("nginx -v")
print(f"Nginx version: {output.decode('utf-8')}")

# Get container stats
stats = container.stats(stream=False)
cpu_delta = stats['cpu_stats']['cpu_usage']['total_usage'] - \
            stats['precpu_stats']['cpu_usage']['total_usage']
system_delta = stats['cpu_stats']['system_cpu_usage'] - \
               stats['precpu_stats']['system_cpu_usage']
cpu_percent = (cpu_delta / system_delta) * stats['cpu_stats']['online_cpus'] * 100
memory_mb = stats['memory_stats']['usage'] / (1024 * 1024)
print(f"CPU: {cpu_percent:.2f}%, Memory: {memory_mb:.1f} MB")

# Stop and remove
container.stop(timeout=10)
container.remove()
print("Container stopped and removed")
```

```python
# docker_image_ops.py — Image operations with docker-py
import docker

client = docker.from_env()

# --- Image Operations ---

# Pull an image
image = client.images.pull('python', tag='3.12-slim')
print(f"Pulled: {image.tags}, Size: {image.attrs['Size'] // (1024*1024)} MB")

# List images
for img in client.images.list():
    tags = img.tags if img.tags else ['<none>']
    size_mb = img.attrs['Size'] // (1024 * 1024)
    print(f"  {tags[0]:40s} | {size_mb:>6d} MB")

# Build an image from a Dockerfile
image, build_logs = client.images.build(
    path="./my-app",
    tag="myapp:latest",
    dockerfile="Dockerfile",
    buildargs={"VERSION": "1.0.0"},
    rm=True,  # Remove intermediate containers
    pull=True  # Always pull base image
)

for log in build_logs:
    if 'stream' in log:
        print(log['stream'], end='')

print(f"\nBuilt: {image.tags}")

# Tag and push
image.tag('registry.acme.io/platform/myapp', tag='v1.0.0')
push_logs = client.images.push(
    'registry.acme.io/platform/myapp',
    tag='v1.0.0',
    stream=True,
    decode=True
)
for log in push_logs:
    print(log)

# Inspect image layers
history = image.history()
for layer in history:
    size_mb = layer['Size'] / (1024 * 1024)
    cmd = layer['CreatedBy'][:80] if layer['CreatedBy'] else '<missing>'
    print(f"  {size_mb:>8.1f} MB | {cmd}")

# Prune unused images
pruned = client.images.prune(filters={'dangling': True})
print(f"Reclaimed: {pruned['SpaceReclaimed'] // (1024*1024)} MB")
```

### Go SDK

```go
// main.go — Docker operations with the Go SDK
package main

import (
    "context"
    "fmt"
    "io"
    "os"

    "github.com/docker/docker/api/types"
    "github.com/docker/docker/api/types/container"
    "github.com/docker/docker/client"
    "github.com/docker/go-connections/nat"
)

func main() {
    ctx := context.Background()

    // Create a Docker client
    cli, err := client.NewClientWithOpts(client.FromEnv,
        client.WithAPIVersionNegotiation())
    if err != nil {
        panic(err)
    }
    defer cli.Close()

    // List containers
    containers, err := cli.ContainerList(ctx, container.ListOptions{All: true})
    if err != nil {
        panic(err)
    }

    fmt.Printf("%-12s %-20s %-10s %s\n", "ID", "NAME", "STATE", "IMAGE")
    for _, c := range containers {
        name := ""
        if len(c.Names) > 0 {
            name = c.Names[0][1:] // Remove leading "/"
        }
        fmt.Printf("%-12s %-20s %-10s %s\n",
            c.ID[:12], name, c.State, c.Image)
    }

    // Create and start a container
    resp, err := cli.ContainerCreate(ctx,
        &container.Config{
            Image: "nginx:alpine",
            Env:   []string{"NGINX_HOST=localhost"},
            ExposedPorts: nat.PortSet{
                "80/tcp": struct{}{},
            },
        },
        &container.HostConfig{
            PortBindings: nat.PortMap{
                "80/tcp": []nat.PortBinding{
                    {HostIP: "0.0.0.0", HostPort: "9090"},
                },
            },
            Resources: container.Resources{
                Memory:   256 * 1024 * 1024, // 256 MB
                CPUQuota: 50000,
            },
        },
        nil, nil, "my-nginx-go")

    if err != nil {
        panic(err)
    }

    if err := cli.ContainerStart(ctx, resp.ID,
        container.StartOptions{}); err != nil {
        panic(err)
    }

    fmt.Printf("Started container: %s\n", resp.ID[:12])

    // Get container logs
    logReader, err := cli.ContainerLogs(ctx, resp.ID,
        container.LogsOptions{ShowStdout: true, Tail: "10"})
    if err != nil {
        panic(err)
    }
    defer logReader.Close()
    io.Copy(os.Stdout, logReader)
}
```

### Node.js SDK (Dockerode)

```javascript
// docker-ops.js — Docker operations with Dockerode
const Docker = require('dockerode');

const docker = new Docker({ socketPath: '/var/run/docker.sock' });

async function main() {
  // List containers
  const containers = await docker.listContainers({ all: true });
  console.log('Containers:');
  containers.forEach(c => {
    console.log(`  ${c.Id.substring(0, 12)} | ${c.Names[0]} | ${c.State} | ${c.Image}`);
  });

  // Run a container
  const container = await docker.createContainer({
    Image: 'nginx:alpine',
    name: 'my-nginx-node',
    ExposedPorts: { '80/tcp': {} },
    HostConfig: {
      PortBindings: { '80/tcp': [{ HostPort: '9091' }] },
      Memory: 256 * 1024 * 1024
    },
    Env: ['NGINX_HOST=localhost'],
    Labels: { app: 'test', 'managed-by': 'nodejs-sdk' }
  });

  await container.start();
  console.log(`Started: ${container.id.substring(0, 12)}`);

  // Get stats
  const stats = await container.stats({ stream: false });
  const memUsageMB = stats.memory_stats.usage / (1024 * 1024);
  console.log(`Memory: ${memUsageMB.toFixed(1)} MB`);

  // Stream logs
  const logStream = await container.logs({
    follow: true,
    stdout: true,
    stderr: true,
    tail: 10
  });
  logStream.pipe(process.stdout);

  // Cleanup after 5 seconds
  setTimeout(async () => {
    await container.stop();
    await container.remove();
    console.log('Container stopped and removed');
    process.exit(0);
  }, 5000);
}

main().catch(console.error);
```

## Building Tools with the Docker API

### Custom Container Dashboard

```python
# dashboard.py — Simple container status dashboard
import docker
import time
import os
from datetime import datetime

client = docker.from_env()

def clear_screen():
    os.system('clear' if os.name == 'posix' else 'cls')

def format_bytes(bytes_val):
    for unit in ['B', 'KB', 'MB', 'GB']:
        if bytes_val < 1024:
            return f"{bytes_val:.1f} {unit}"
        bytes_val /= 1024
    return f"{bytes_val:.1f} TB"

def get_container_stats(container):
    """Get CPU and memory stats for a container."""
    try:
        stats = container.stats(stream=False)

        # CPU calculation
        cpu_delta = stats['cpu_stats']['cpu_usage']['total_usage'] - \
                    stats['precpu_stats']['cpu_usage']['total_usage']
        system_delta = stats['cpu_stats']['system_cpu_usage'] - \
                       stats['precpu_stats']['system_cpu_usage']
        num_cpus = stats['cpu_stats'].get('online_cpus',
                     len(stats['cpu_stats']['cpu_usage'].get('percpu_usage', [1])))

        cpu_percent = 0.0
        if system_delta > 0 and cpu_delta > 0:
            cpu_percent = (cpu_delta / system_delta) * num_cpus * 100

        # Memory calculation
        mem_usage = stats['memory_stats'].get('usage', 0)
        mem_limit = stats['memory_stats'].get('limit', 0)
        mem_percent = (mem_usage / mem_limit * 100) if mem_limit > 0 else 0

        return {
            'cpu_percent': cpu_percent,
            'mem_usage': mem_usage,
            'mem_limit': mem_limit,
            'mem_percent': mem_percent,
        }
    except Exception:
        return None

def display_dashboard():
    """Display the container dashboard."""
    clear_screen()

    info = client.info()
    print("=" * 78)
    print(f"  Docker Dashboard — {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    print(f"  Host: {info.get('Name', 'unknown')} | "
          f"Docker {info['ServerVersion']} | "
          f"Containers: {info['ContainersRunning']}/{info['Containers']} | "
          f"Images: {info['Images']}")
    print("=" * 78)
    print()

    containers = client.containers.list(all=True)

    if not containers:
        print("  No containers found.")
        return

    # Header
    print(f"  {'NAME':<25} {'IMAGE':<25} {'STATUS':<15} {'CPU':>6} {'MEM':>10} {'PORTS'}")
    print(f"  {'-'*25} {'-'*25} {'-'*15} {'-'*6} {'-'*10} {'-'*20}")

    for container in sorted(containers, key=lambda c: c.name):
        image = container.image.tags[0] if container.image.tags else 'unknown'
        image = image[:24]  # Truncate long image names

        # Status with color indicator
        status = container.status
        if status == 'running':
            health = container.attrs.get('State', {}).get('Health', {})
            if health:
                health_status = health.get('Status', 'none')
                status = f"UP ({health_status})"
            else:
                status = "UP"
        elif status == 'exited':
            exit_code = container.attrs['State']['ExitCode']
            status = f"EXIT ({exit_code})"

        # Get resource stats for running containers
        cpu_str = "-"
        mem_str = "-"
        if container.status == 'running':
            stats = get_container_stats(container)
            if stats:
                cpu_str = f"{stats['cpu_percent']:.1f}%"
                mem_str = format_bytes(stats['mem_usage'])

        # Get port mappings
        ports = container.attrs['NetworkSettings']['Ports'] or {}
        port_str = ""
        for container_port, host_bindings in ports.items():
            if host_bindings:
```

### Automated Cleanup Tool

```python
# cleanup.py — Automated Docker resource cleanup
import docker
from datetime import datetime, timezone, timedelta

client = docker.from_env()

def cleanup_stopped_containers(max_age_hours=24):
    """Remove containers that have been stopped for more than max_age_hours."""
    print(f"\n--- Stopped Containers (older than {max_age_hours}h) ---")
    removed = 0

    for container in client.containers.list(all=True,
                                             filters={'status': 'exited'}):
        finished = container.attrs['State']['FinishedAt']
        finished_dt = datetime.fromisoformat(finished.replace('Z', '+00:00'))
        age = datetime.now(timezone.utc) - finished_dt

        if age > timedelta(hours=max_age_hours):
            name = container.name
            exit_code = container.attrs['State']['ExitCode']
            print(f"  Removing: {name} (exited {exit_code}, "
                  f"stopped {age.days}d {age.seconds//3600}h ago)")
            container.remove(force=True)
            removed += 1

    print(f"  Removed {removed} containers")
    return removed

def cleanup_dangling_images():
    """Remove dangling (untagged) images."""
    print("\n--- Dangling Images ---")
    result = client.images.prune(filters={'dangling': True})
    deleted = result.get('ImagesDeleted') or []
    space = result.get('SpaceReclaimed', 0)
    print(f"  Removed {len(deleted)} images, reclaimed "
          f"{space // (1024*1024)} MB")
    return len(deleted)

def cleanup_unused_volumes():
    """Remove volumes not attached to any container."""
    print("\n--- Unused Volumes ---")
    result = client.volumes.prune()
    deleted = result.get('VolumesDeleted') or []
    space = result.get('SpaceReclaimed', 0)
    print(f"  Removed {len(deleted)} volumes, reclaimed "
          f"{space // (1024*1024)} MB")
    return len(deleted)

def cleanup_unused_networks():
    """Remove networks not used by any container."""
    print("\n--- Unused Networks ---")
    result = client.networks.prune()
    deleted = result.get('NetworksDeleted') or []
    print(f"  Removed {len(deleted)} networks")
    return len(deleted)

if __name__ == '__main__':
    print(f"Docker Cleanup — {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    print("=" * 50)

    cleanup_stopped_containers(max_age_hours=48)
    cleanup_dangling_images()
    cleanup_unused_volumes()
    cleanup_unused_networks()

    print("\n" + "=" * 50)
    print("Cleanup complete.")
```

### Event-Driven Monitoring with Python

```python
# event_monitor.py — Real-time Docker event monitor with alerting
import docker
import json
from datetime import datetime

client = docker.from_env()

# Events that indicate problems
ALERT_EVENTS = {
    'die': 'Container died unexpectedly',
    'oom': 'Container killed by OOM (Out of Memory)',
    'kill': 'Container was killed',
    'health_status: unhealthy': 'Container health check failing',
}

def format_event(event):
    """Format a Docker event for display."""
    timestamp = datetime.fromtimestamp(event['time']).strftime('%H:%M:%S')
    action = event['Action']
    actor = event['Actor']
    name = actor['Attributes'].get('name', 'unknown')
    image = actor['Attributes'].get('image', 'unknown')
    event_type = event['Type']

    return f"[{timestamp}] {event_type}/{action}: {name} ({image})"

def send_alert(event, reason):
    """Send alert for critical events."""
    actor = event['Actor']
    name = actor['Attributes'].get('name', 'unknown')

    alert = {
        'severity': 'critical',
        'container': name,
        'reason': reason,
        'image': actor['Attributes'].get('image', 'unknown'),
        'exit_code': actor['Attributes'].get('exitCode', 'N/A'),
        'timestamp': datetime.fromtimestamp(event['time']).isoformat(),
    }

    print(f"\n  *** ALERT: {reason} ***")
    print(f"  Container: {name}")
    print(f"  Exit Code: {alert['exit_code']}")
    print(f"  Image: {alert['image']}")

    # In production, send to Slack, PagerDuty, etc.
    # requests.post(SLACK_WEBHOOK, json={'text': json.dumps(alert)})

def monitor_events():
    """Stream and process Docker events in real time."""
    print("Docker Event Monitor")
    print("=" * 60)
    print(f"Started at: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    print("Monitoring for: container, image, network, volume events")
    print("=" * 60)

    # Filter for specific event types
    filters = {
        'type': ['container', 'image', 'network', 'volume'],
    }

    for event in client.events(decode=True, filters=filters):
        line = format_event(event)
        print(line)

        # Check for alert-worthy events
        action = event.get('Action', '')
        for trigger, reason in ALERT_EVENTS.items():
            if trigger in action:
                send_alert(event, reason)

if __name__ == '__main__':
    try:
        monitor_events()
    except KeyboardInterrupt:
        print("\nMonitor stopped.")
```

## API Versioning and Compatibility

The Docker API version changes with each Docker release. Your tools need to handle version differences gracefully.

```bash

curl -s --unix-socket /var/run/docker.sock http://localhost/v1.43/containers/json | jq length

# Version negotiation in Python

# Pin to a specific version for stability
```


<Callout type="warning" title="Remote API Security">
If you expose the Docker API over TCP for remote access, you **must** configure TLS. An unprotected Docker API on the network is equivalent to giving anyone root access to your server.

```bash
# NEVER do this in production:
# dockerd -H tcp://0.0.0.0:2375  # Unencrypted, unauthenticated

# ALWAYS use TLS:
dockerd --tlsverify \
  --tlscacert=ca.pem \
  --tlscert=server-cert.pem \
  --tlskey=server-key.pem \
  -H tcp://0.0.0.0:2376

# Or use SSH tunneling (simpler and secure):
docker -H ssh://user@remote-host ps
```

## Key Takeaways

- **The Docker CLI is just an API client** — every `docker` command maps to a REST API call over the Unix socket at `/var/run/docker.sock`
- **curl with --unix-socket** is the simplest way to explore the Docker API — great for debugging and understanding request/response formats
- **Container operations via API** follow a lifecycle: create, start, (inspect, logs, stats, exec), stop, remove
- **Docker SDKs** (Python docker-py, Go docker/client, Node.js dockerode) provide language-native interfaces that are much easier to use than raw HTTP
- **Event streaming** enables real-time monitoring — subscribe to container lifecycle events to build dashboards and alerting systems
- **API versioning** matters — always specify the API version in production code to avoid breaking changes when Docker is updated
- **The Docker socket is root-equivalent** — treat access to `/var/run/docker.sock` as a security-critical permission
- **Custom tooling** built on the Docker API can automate cleanup, monitoring, health management, and reporting tasks that are tedious with the CLI
