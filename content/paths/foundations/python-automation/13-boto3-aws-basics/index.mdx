## Setting Up Boto3

### Installation and Credentials

```python
# pip install boto3
import boto3

# boto3 looks for credentials in this order:
# 1. Explicit parameters (boto3.Session(aws_access_key_id=...))
# 2. Environment variables (AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY)
# 3. Shared credentials file (~/.aws/credentials)
# 4. AWS config file (~/.aws/config)
# 5. Instance metadata (EC2 instance role, ECS task role, Lambda role)
```

### Using Profiles

```python
import boto3

# Default profile
session = boto3.Session()

# Named profile from ~/.aws/credentials
dev_session = boto3.Session(profile_name="dev")
prod_session = boto3.Session(profile_name="production")

# Explicit region
session = boto3.Session(region_name="us-east-1")

# Create clients from a session
ec2 = session.client("ec2")
s3 = session.client("s3")
```

### Environment-Based Configuration

```python
import boto3
import os

def get_aws_session():
    """Create a boto3 session from environment or profile."""
    profile = os.getenv("AWS_PROFILE")
    region = os.getenv("AWS_DEFAULT_REGION", "us-east-1")

    if profile:
        return boto3.Session(profile_name=profile, region_name=region)

    # Fall back to environment variables or instance role
    return boto3.Session(region_name=region)

session = get_aws_session()
```

<Callout type="tip" title="Never Hardcode AWS Credentials">
Use IAM roles for EC2 instances, ECS tasks, and Lambda functions. Use profiles for local development. Use environment variables in CI/CD. Never put access keys in source code — even in private repos, credentials leak through git history.
</Callout>

## Client vs Resource APIs

Boto3 offers two interfaces to AWS services:

```python
import boto3

# Client API — low-level, maps 1:1 to AWS API calls
# Returns raw dictionaries
ec2_client = boto3.client("ec2")
response = ec2_client.describe_instances()
# response is a dict with 'Reservations' key

# Resource API — high-level, object-oriented
# Returns Python objects with attributes and methods
ec2_resource = boto3.resource("ec2")
instances = ec2_resource.instances.all()
# instances is an iterable of Instance objects

# When to use which:
# Client: full API coverage, pagination control, batch operations
# Resource: cleaner code, simpler iteration, but not all services supported
```

### Using the Resource API

```python
import boto3

ec2 = boto3.resource("ec2")

# Iterate all instances (cleaner than client API)
for instance in ec2.instances.all():
    name = ""
    for tag in (instance.tags or []):
        if tag["Key"] == "Name":
            name = tag["Value"]
    print(f"  {instance.id}: {name} ({instance.state['Name']})")

running = ec2.instances.filter(
```

## Error Handling

AWS errors come through `botocore.exceptions`:

```python
import boto3
from botocore.exceptions import ClientError, NoCredentialsError

ec2 = boto3.client("ec2")

try:
    response = ec2.describe_instances(InstanceIds=["i-nonexistent"])
except ClientError as e:
    error_code = e.response["Error"]["Code"]
    error_msg = e.response["Error"]["Message"]

    if error_code == "InvalidInstanceID.NotFound":
        print(f"Instance not found: {error_msg}")
    elif error_code == "UnauthorizedOperation":
        print(f"Permission denied: {error_msg}")
    else:
        print(f"AWS error ({error_code}): {error_msg}")

except NoCredentialsError:
    print("No AWS credentials found. Configure with 'aws configure'")
```

<Callout type="warning" title="Always Catch ClientError">
Every boto3 call can raise `ClientError`. The error code tells you what went wrong — `InvalidInstanceID.NotFound`, `AccessDenied`, `ThrottlingException`, etc. Always catch it and handle the specific codes your script might encounter.
</Callout>

## Pagination

Many AWS APIs return paginated results. Use paginators to handle this automatically:

```python
import boto3

s3 = boto3.client("s3")

# Manual pagination (tedious)
all_objects = []
continuation_token = None

while True:
    kwargs = {"Bucket": "my-bucket", "MaxKeys": 1000}
    if continuation_token:
        kwargs["ContinuationToken"] = continuation_token

    response = s3.list_objects_v2(**kwargs)
    all_objects.extend(response.get("Contents", []))

    if not response.get("IsTruncated"):
        break
    continuation_token = response["NextContinuationToken"]

# Using a paginator (much better)
paginator = s3.get_paginator("list_objects_v2")
all_objects = []

for page in paginator.paginate(Bucket="my-bucket"):
    for obj in page.get("Contents", []):
        all_objects.append(obj)

print(f"Total objects: {len(all_objects)}")
```

## Waiters

Wait for AWS resources to reach a specific state:

```python
import boto3

ec2 = boto3.client("ec2")

# Start an instance
instance_ids = ["i-0abc123def456789"]
ec2.start_instances(InstanceIds=instance_ids)

# Wait for it to be running (polls every 15s, up to 40 attempts)
print("Waiting for instance to be running...")
waiter = ec2.get_waiter("instance_running")
waiter.wait(
    InstanceIds=instance_ids,
    WaiterConfig={"Delay": 10, "MaxAttempts": 30}
)
print("Instance is now running!")
```

Build a tool that analyzes S3 buckets for size, object count, and configuration.

```python
import boto3
from botocore.exceptions import ClientError, NoCredentialsError
from datetime import datetime, timezone

def analyze_s3_buckets():
    """Analyze all S3 buckets in the account."""
    try:
        s3 = boto3.client("s3")
        response = s3.list_buckets()
        buckets = response["Buckets"]
    except (NoCredentialsError, ClientError):
        print("No AWS credentials. Using mock data.\n")
        buckets = None

    if not buckets:
        return analyze_mock_buckets()

    results = []
    for bucket in buckets:
        name = bucket["Name"]
        info = {"name": name, "created": bucket["CreationDate"]}

        # Get bucket location
        try:
            loc = s3.get_bucket_location(Bucket=name)
            info["region"] = loc["LocationConstraint"] or "us-east-1"
        except ClientError:
            info["region"] = "unknown"

        # Count objects and total size
        try:
            paginator = s3.get_paginator("list_objects_v2")
            total_size = 0
            total_objects = 0
            for page in paginator.paginate(Bucket=name):
                for obj in page.get("Contents", []):
                    total_size += obj["Size"]
                    total_objects += 1
            info["object_count"] = total_objects
            info["total_size_bytes"] = total_size
        except ClientError:
            info["object_count"] = "?"
            info["total_size_bytes"] = 0

        # Check versioning
        try:
            versioning = s3.get_bucket_versioning(Bucket=name)
            info["versioning"] = versioning.get("Status", "Disabled")
        except ClientError:
            info["versioning"] = "unknown"

        results.append(info)

    return results

def analyze_mock_buckets():

         "object_count": 89000, "total_size_bytes": 8_000_000_000,

         "object_count": 45, "total_size_bytes": 5_000_000,

def format_size(bytes_val):
    """Format bytes as human-readable size."""
    for unit in ("B", "KB", "MB", "GB", "TB"):
        if bytes_val < 1024:
            return f"{bytes_val:.1f} {unit}"
        bytes_val /= 1024
    return f"{bytes_val:.1f} PB"

def print_s3_report(buckets):
    """Print formatted S3 analysis."""
    print("=" * 70)
    print("  S3 BUCKET ANALYSIS")
    print("=" * 70)

    total_size = sum(b.get("total_size_bytes", 0) for b in buckets)
    total_objects = sum(
        b["object_count"] for b in buckets
        if isinstance(b.get("object_count"), int)
    )

    print(f"\n  Total buckets:  {len(buckets)}")
    print(f"  Total objects:  {total_objects:,}")
    print(f"  Total size:     {format_size(total_size)}")

    # Bucket table
    print(f"\n  {'Bucket':<30} {'Region':<15} {'Objects':>10} {'Size':>12} {'Versioning'}")
    print(f"  {'-'*30} {'-'*15} {'-'*10} {'-'*12} {'-'*10}")

    for b in sorted(buckets, key=lambda x: x.get("total_size_bytes", 0),
                    reverse=True):
        obj_count = (f"{b['object_count']:,}"
                     if isinstance(b.get("object_count"), int) else "?")
        size = format_size(b.get("total_size_bytes", 0))
        ver = b.get("versioning", "?")
        print(f"  {b['name']:<30} {b.get('region', '?'):<15} "
              f"{obj_count:>10} {size:>12} {ver}")

    # Warnings
    no_versioning = [b for b in buckets if b.get("versioning") == "Disabled"]
    if no_versioning:
        print(f"\n  WARNING: {len(no_versioning)} bucket(s) without versioning:")
        for b in no_versioning:
            print(f"    - {b['name']}")

    print("\n" + "=" * 70)

# --- Run ---
buckets = analyze_s3_buckets()
print_s3_report(buckets)
```

```python
        """Get S3 bucket summary."""
        if self.use_mock:
            return self._mock_s3()

        s3 = self._client("s3")
        buckets = []
        for bucket in s3.list_buckets()["Buckets"]:
            info = {
                "name": bucket["Name"],
                "created": bucket["CreationDate"].isoformat(),
            }
            try:
                loc = s3.get_bucket_location(Bucket=bucket["Name"])
                info["region"] = loc["LocationConstraint"] or "us-east-1"
            except ClientError:
                info["region"] = "unknown"
            buckets.append(info)
        return buckets

        return [
            {"name": "prod-app-assets", "region": "us-east-1",

    def get_iam_summary(self):
        """Get IAM users summary."""
        if self.use_mock:
            return self._mock_iam()

        iam = self._client("iam")
        users = []
        for user in iam.list_users()["Users"]:

                "username": user["UserName"],
                "created": user["CreateDate"].isoformat(),
                "has_mfa": False,
                "access_keys": 0,

            try:
                mfa = iam.list_mfa_devices(UserName=user["UserName"])
                info["has_mfa"] = len(mfa["MFADevices"]) > 0
            except ClientError:
                pass
            try:
                keys = iam.list_access_keys(UserName=user["UserName"])

    # --- Full Report ---
    def generate_report(self):
        """Generate a complete infrastructure report."""
        print("Collecting AWS infrastructure data...")
        ec2_data = self.get_ec2_summary()
        s3_data = self.get_s3_summary()
        iam_data = self.get_iam_summary()

        report = {
            "generated_at": datetime.now(timezone.utc).isoformat(),
            "region": self.region,
            "ec2": ec2_data,
            "s3": s3_data,
            "iam": iam_data,

    def _print_report(self, report):
        """Print the full formatted report."""
        print()
        print("=" * 65)
        print("  AWS INFRASTRUCTURE REPORT")
        print(f"  Region: {report['region']}")
        print(f"  Generated: {report['generated_at'][:19]} UTC")
        print("=" * 65)

        # EC2 Section
        ec2 = report["ec2"]
        by_state = defaultdict(int)
        by_env = defaultdict(int)
        by_type = defaultdict(int)
        for inst in ec2:
            by_state[inst["state"]] += 1
            by_env[inst["environment"]] += 1
            by_type[inst["type"]] += 1

        print(f"\n  --- EC2 Instances ({len(ec2)}) ---")
        print(f"  By State:       {dict(by_state)}")
        print(f"  By Environment: {dict(by_env)}")
        print(f"  By Type:        {dict(by_type)}")

        print(f"\n  {'ID':<12} {'Name':<15} {'Type':<12} {'State':<10} {'Env'}")
        for inst in ec2:
            print(f"  {inst['id']:<12} {inst['name']:<15} "
                  f"{inst['type']:<12} {inst['state']:<10} "
                  f"{inst['environment']}")

        # S3 Section
        s3 = report["s3"]
        print(f"\n  --- S3 Buckets ({len(s3)}) ---")
        for bucket in s3:
            print(f"  {bucket['name']:<35} {bucket['region']:<15} "
                  f"Created: {str(bucket['created'])[:10]}")

        # IAM Section
        iam = report["iam"]
        print(f"\n  --- IAM Users ({len(iam)}) ---")
        no_mfa = [u for u in iam if not u["has_mfa"]]
        print(f"  Users without MFA: {len(no_mfa)}")

        for user in iam:
            mfa_status = "MFA" if user["has_mfa"] else "NO MFA"
            print(f"  {user['username']:<20} Keys: {user['access_keys']}  "
                  f"[{mfa_status}]")

        # Security Warnings
        print(f"\n  --- Security Warnings ---")
        warnings = []
        untagged = [i for i in ec2 if i["environment"] == "untagged"]
        if untagged:
            warnings.append(
                f"{len(untagged)} EC2 instance(s) without Environment tag"
            )
        if no_mfa:
            warnings.append(
                f"{len(no_mfa)} IAM user(s) without MFA: "
                f"{', '.join(u['username'] for u in no_mfa)}"
            )
        stopped = [i for i in ec2 if i["state"] == "stopped"]
        if stopped:
            warnings.append(
                f"{len(stopped)} stopped instance(s) (EBS costs continue)"
            )

        if warnings:
            for w in warnings:
                print(f"  [!] {w}")
        else:
            print("  No warnings.")

        print("\n" + "=" * 65)
```

## Key Takeaways

- boto3 finds credentials automatically: environment variables, profiles, or IAM roles — never hardcode keys
- Use the **client** API for full control and pagination; use the **resource** API for cleaner iteration
- Always catch `ClientError` and check `e.response["Error"]["Code"]` for specific AWS error handling
- Use **paginators** (`client.get_paginator()`) to handle multi-page AWS API responses automatically
- Use **waiters** (`client.get_waiter()`) to wait for resources to reach a desired state
- Filter EC2 instances with the `Filters` parameter to reduce API calls and response size
- S3 `generate_presigned_url()` creates temporary download links without making objects public
