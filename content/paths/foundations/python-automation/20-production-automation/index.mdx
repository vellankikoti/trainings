<Callout type="story" title="The Story">
Your automation scripts work perfectly on your laptop. Then they hit production. The config file path is different. The API key is hardcoded. Running the script twice creates duplicate resources. It cannot handle two servers at once because it runs sequentially. There are no logs, no error messages, and when it fails at 3 AM, nobody knows what happened. You refactor using production patterns: config files, secrets from a vault, idempotent operations, concurrent execution, structured logging, and proper packaging. The script becomes a tool. The tool becomes reliable. You sleep through the night.
</Callout>

## Configuration Management

Production automation separates configuration from code. The configuration hierarchy (with later sources overriding earlier ones) is: defaults, config file, environment variables, command-line arguments.

### Configuration from Multiple Sources

```python
import os
import json
import yaml
from pathlib import Path
from dataclasses import dataclass, field
from typing import Optional


@dataclass
class AppConfig:
    """Application configuration with defaults."""
    app_name: str = "devops-tool"
    environment: str = "development"
    log_level: str = "INFO"
    database_url: str = ""
    redis_url: str = ""
    api_base_url: str = "http://localhost:8080"
    workers: int = 4
    timeout: int = 30
    dry_run: bool = False
    debug: bool = False


def load_config(config_path=None, overrides=None):
    """Load configuration from defaults, file, env vars, and overrides."""
    config = AppConfig()

    # 1. Load from config file (if provided)
    if config_path:
        path = Path(config_path)
        if path.exists():
            raw = path.read_text()
            if path.suffix in (".yaml", ".yml"):
                file_config = yaml.safe_load(raw) or {}
            elif path.suffix == ".json":
                file_config = json.loads(raw)
            else:
                raise ValueError(f"Unsupported config format: {path.suffix}")

            for key, value in file_config.items():
                if hasattr(config, key):
                    setattr(config, key, value)

    # 2. Override with environment variables (prefixed with APP_)
    env_mapping = {
        "APP_NAME": "app_name",
        "APP_ENV": "environment",
        "APP_LOG_LEVEL": "log_level",
        "DATABASE_URL": "database_url",
        "REDIS_URL": "redis_url",
        "APP_API_URL": "api_base_url",
        "APP_WORKERS": "workers",
        "APP_TIMEOUT": "timeout",
        "APP_DRY_RUN": "dry_run",
        "APP_DEBUG": "debug",
    }

    for env_var, attr_name in env_mapping.items():
        value = os.environ.get(env_var)
        if value is not None:
            # Type conversion based on the default value type
            current = getattr(config, attr_name)
            if isinstance(current, bool):
                setattr(config, attr_name, value.lower() in ("true", "1", "yes"))
            elif isinstance(current, int):
                setattr(config, attr_name, int(value))
            else:
                setattr(config, attr_name, value)

    # 3. Override with explicit overrides (from CLI args)
    if overrides:
        for key, value in overrides.items():
            if hasattr(config, key):
                setattr(config, key, value)

    return config


# Usage:
# config = load_config("config.yaml", overrides={"dry_run": True})
# config = load_config()  # defaults + env vars only
```

## Secrets Handling

The cardinal rule of secrets: never in code, never in version control, never in logs.

### Environment Variables and .env Files

```python
import os
from pathlib import Path


def load_dotenv(env_file=".env"):
    """Simple .env file loader (or use python-dotenv package)."""
    path = Path(env_file)
    if not path.exists():
        return

    for line in path.read_text().splitlines():
        line = line.strip()
        if not line or line.startswith("#"):
            continue
        if "=" in line:
            key, _, value = line.partition("=")
            key = key.strip()
            value = value.strip().strip("'\"")
            os.environ.setdefault(key, value)  # Don't override existing env vars


class SecretManager:
    """Load secrets from environment variables with validation."""

    def __init__(self, prefix=""):
        self.prefix = prefix

    def get(self, name, required=True):
        """Get a secret from environment variables."""
        env_var = f"{self.prefix}{name}" if self.prefix else name
        value = os.environ.get(env_var)

        if required and not value:
            raise EnvironmentError(
                f"Required secret '{env_var}' is not set. "
                f"Set it with: export {env_var}=<value>"
            )
        return value

    def get_all(self, names):
        """Get multiple secrets, raising an error if any are missing."""
        missing = []
        secrets = {}
        for name in names:
            env_var = f"{self.prefix}{name}" if self.prefix else name
            value = os.environ.get(env_var)
            if value:
                secrets[name] = value
            else:
                missing.append(env_var)

        if missing:
            raise EnvironmentError(
                f"Missing required secrets: {', '.join(missing)}\n"
                f"Set them in your environment or .env file."
            )
        return secrets


# Usage:
# load_dotenv(".env")
# secrets = SecretManager(prefix="APP_")
# db_password = secrets.get("DATABASE_PASSWORD")
# all_secrets = secrets.get_all(["API_KEY", "DATABASE_PASSWORD", "SLACK_WEBHOOK"])
```

### HashiCorp Vault

```python
# pip install hvac

import hvac
import os


class VaultClient:
    """Simple HashiCorp Vault client for secret retrieval."""

    def __init__(self, url=None, token=None):
        self.client = hvac.Client(
            url=url or os.environ.get("VAULT_ADDR", "http://localhost:8200"),
            token=token or os.environ.get("VAULT_TOKEN"),
        )
        if not self.client.is_authenticated():
            raise RuntimeError("Vault authentication failed")

    def get_secret(self, path, mount_point="secret"):
        """Read a secret from Vault KV v2."""
        result = self.client.secrets.kv.v2.read_secret_version(
            path=path,
            mount_point=mount_point,
        )
        return result["data"]["data"]

    def get_database_creds(self, role):
        """Get dynamic database credentials from Vault."""
        result = self.client.secrets.database.generate_credentials(name=role)
        return {
            "username": result["data"]["username"],
            "password": result["data"]["password"],
            "ttl": result["lease_duration"],
        }


# Usage:
# vault = VaultClient()
# secrets = vault.get_secret("app/production")
# db_creds = vault.get_database_creds("app-readonly")
```

<Callout type="warning" title="Never Log Secrets">
Secrets will inevitably end up in variables, function arguments, and error messages. Implement a secret redactor in your logging setup that replaces known secret patterns (API keys, passwords, tokens) with `***`. A single logged database password can compromise your entire infrastructure.
</Callout>

## Idempotency

Idempotent operations produce the same result whether you run them once or a hundred times. This is critical for automation that might be retried, run by cron, or triggered by multiple events.

### Check-Before-Act Pattern

```python
import os
import subprocess


def ensure_user_exists(username, groups=None):
    """Ensure a system user exists (idempotent)."""
    # Check current state
    try:
        result = subprocess.run(
            ["id", username], capture_output=True, text=True
        )
        if result.returncode == 0:
            print(f"  User '{username}' already exists")
            return False  # No change made
    except FileNotFoundError:
        pass

    # Act: create the user
    cmd = ["useradd", "-m", username]
    if groups:
        cmd.extend(["-G", ",".join(groups)])
    subprocess.run(cmd, check=True)
    print(f"  Created user '{username}'")
    return True  # Change made


def ensure_directory(path, mode=0o755, owner=None):
    """Ensure a directory exists with correct permissions (idempotent)."""
    changed = False

    if not os.path.exists(path):
        os.makedirs(path, mode=mode)
        print(f"  Created directory: {path}")
        changed = True
    else:
        current_mode = oct(os.stat(path).st_mode)[-3:]
        desired_mode = oct(mode)[-3:]
        if current_mode != desired_mode:
            os.chmod(path, mode)
            print(f"  Fixed permissions on {path}: {current_mode} -> {desired_mode}")
            changed = True

    if owner:
        import pwd
        uid = pwd.getpwnam(owner).pw_uid
        gid = pwd.getpwnam(owner).pw_gid
        current_uid = os.stat(path).st_uid
        if current_uid != uid:
            os.chown(path, uid, gid)
            changed = True

    if not changed:
        print(f"  Directory OK: {path}")

    return changed
```

### Desired State Pattern

```python
def ensure_desired_state(current_state, desired_state, apply_func):
    """Generic desired-state reconciliation."""
    changes = []

    # Find items to add
    for key, desired in desired_state.items():
        current = current_state.get(key)
        if current is None:
            changes.append(("add", key, desired))
        elif current != desired:
            changes.append(("update", key, desired))

    # Find items to remove
    for key in current_state:
        if key not in desired_state:
            changes.append(("remove", key, None))

    if not changes:
        print("  No changes needed — desired state matches current state")
        return []

    # Apply changes
    results = []
    for action, key, value in changes:
        try:
            apply_func(action, key, value)
            results.append({"action": action, "key": key, "status": "success"})
            print(f"  {action.upper():8s} {key}")
        except Exception as e:
            results.append({"action": action, "key": key, "status": "error", "error": str(e)})
            print(f"  {action.upper():8s} {key} — ERROR: {e}")

    return results
```

## Concurrency

DevOps automation often needs to work on multiple servers, check multiple endpoints, or process multiple files simultaneously. Python offers threading for I/O-bound work and asyncio for high-concurrency I/O.

### ThreadPoolExecutor

```python
from concurrent.futures import ThreadPoolExecutor, as_completed
import requests
import time


def check_server(server):
    """Check a single server (I/O-bound: network call)."""
    name, url = server
    try:
        start = time.time()
        response = requests.get(url, timeout=5)
        elapsed = time.time() - start
        return {
            "name": name,
            "status": response.status_code,
            "time_ms": round(elapsed * 1000),
            "healthy": response.ok,
        }
    except requests.RequestException as e:
        return {"name": name, "healthy": False, "error": str(e)}


def check_all_servers(servers, max_workers=10):
    """Check multiple servers concurrently."""
    results = []
    start = time.time()

    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        # Submit all tasks
        future_to_server = {
            executor.submit(check_server, server): server
            for server in servers
        }

        # Collect results as they complete
        for future in as_completed(future_to_server):
            server = future_to_server[future]
            try:
                result = future.result()
                results.append(result)
            except Exception as e:
                results.append({"name": server[0], "healthy": False, "error": str(e)})

    elapsed = time.time() - start
    print(f"Checked {len(servers)} servers in {elapsed:.1f}s "
          f"(would take ~{len(servers) * 5:.0f}s sequentially)")
    return results


# Usage:
# servers = [(f"server-{i}", f"http://10.0.1.{i}:8080/health") for i in range(50)]
# results = check_all_servers(servers)
```

### asyncio for High-Concurrency I/O

```python
# pip install aiohttp

import asyncio
import aiohttp
import time


async def check_server_async(session, name, url):
    """Check a single server using async HTTP."""
    try:
        start = time.time()
        async with session.get(url, timeout=aiohttp.ClientTimeout(total=5)) as response:
            elapsed = time.time() - start
            return {
                "name": name,
                "status": response.status,
                "time_ms": round(elapsed * 1000),
                "healthy": response.status == 200,
            }
    except Exception as e:
        return {"name": name, "healthy": False, "error": str(e)}


async def check_all_async(servers):
    """Check all servers concurrently with asyncio."""
    start = time.time()

    async with aiohttp.ClientSession() as session:
        tasks = [
            check_server_async(session, name, url)
            for name, url in servers
        ]
        results = await asyncio.gather(*tasks)

    elapsed = time.time() - start
    print(f"Checked {len(servers)} servers in {elapsed:.1f}s")
    return results


# Usage:
# servers = [(f"server-{i}", f"http://10.0.1.{i}:8080/health") for i in range(100)]
# results = asyncio.run(check_all_async(servers))
```

## Rate Limiting and Retry with Tenacity

External APIs have rate limits and transient failures. The `tenacity` library provides sophisticated retry logic:

```python
# pip install tenacity

from tenacity import (
    retry, stop_after_attempt, wait_exponential,
    retry_if_exception_type, before_sleep_log,
)
import requests
import logging

logger = logging.getLogger(__name__)


@retry(
    stop=stop_after_attempt(5),
    wait=wait_exponential(multiplier=1, min=1, max=60),
    retry=retry_if_exception_type((requests.ConnectionError, requests.Timeout)),
    before_sleep=before_sleep_log(logger, logging.WARNING),
)
def call_api(url, payload):
    """Call an API with automatic retry on transient failures."""
    response = requests.post(url, json=payload, timeout=10)
    response.raise_for_status()
    return response.json()


# Retry behavior:
# Attempt 1: immediate
# Attempt 2: wait 1s
# Attempt 3: wait 2s
# Attempt 4: wait 4s
# Attempt 5: wait 8s
# Then give up


@retry(
    stop=stop_after_attempt(3),
    wait=wait_exponential(multiplier=2, min=2, max=30),
    retry=retry_if_exception_type(Exception),
)
def deploy_with_retry(app_name, image):
    """Deploy with retries for transient infrastructure issues."""
    print(f"  Attempting deploy: {app_name} -> {image}")
    # ... deployment logic ...
    return {"status": "success"}
```

## Structured Logging

Production scripts need structured logs — parseable by log aggregation tools (ELK, Datadog, CloudWatch), not just human eyes.

```python
import logging
import json
import sys
import uuid
from datetime import datetime


class JSONFormatter(logging.Formatter):
    """Format log records as JSON for log aggregation."""

    def format(self, record):
        log_entry = {
            "timestamp": datetime.utcnow().isoformat() + "Z",
            "level": record.levelname,
            "logger": record.name,
            "message": record.getMessage(),
            "module": record.module,
            "function": record.funcName,
            "line": record.lineno,
        }

        # Include extra fields
        if hasattr(record, "correlation_id"):
            log_entry["correlation_id"] = record.correlation_id
        if hasattr(record, "server"):
            log_entry["server"] = record.server
        if hasattr(record, "duration_ms"):
            log_entry["duration_ms"] = record.duration_ms

        # Include exception info
        if record.exc_info and record.exc_info[0]:
            log_entry["exception"] = {
                "type": record.exc_info[0].__name__,
                "message": str(record.exc_info[1]),
            }

        return json.dumps(log_entry)


def setup_logging(level="INFO", json_output=True):
    """Set up structured logging."""
    root_logger = logging.getLogger()
    root_logger.setLevel(getattr(logging, level))

    handler = logging.StreamHandler(sys.stdout)

    if json_output:
        handler.setFormatter(JSONFormatter())
    else:
        handler.setFormatter(logging.Formatter(
            "%(asctime)s [%(levelname)s] %(name)s: %(message)s"
        ))

    root_logger.addHandler(handler)
    return root_logger


# Correlation IDs for tracing operations across log lines
class CorrelationContext:
    """Add a correlation ID to all log messages in a block."""

    def __init__(self, logger, operation_name=""):
        self.logger = logger
        self.correlation_id = str(uuid.uuid4())[:8]
        self.operation = operation_name

    def __enter__(self):
        self.logger.info(
            f"Starting: {self.operation}",
            extra={"correlation_id": self.correlation_id},
        )
        return self

    def __exit__(self, *args):
        self.logger.info(
            f"Completed: {self.operation}",
            extra={"correlation_id": self.correlation_id},
        )

    def log(self, level, message, **extra):
        extra["correlation_id"] = self.correlation_id
        getattr(self.logger, level)(message, extra=extra)


# Usage:
# setup_logging(level="INFO", json_output=True)
# logger = logging.getLogger("deploy")
#
# with CorrelationContext(logger, "deploy-web-app") as ctx:
#     ctx.log("info", "Pulling image", server="web-01")
#     ctx.log("info", "Starting containers", server="web-01", duration_ms=1200)
```

```python
SECRET_PATTERNS = {"password", "secret", "token", "key", "credential", "api_key"}


def redact_value(key, value):
    """Redact a value if its key suggests it is a secret."""
    key_lower = key.lower()
    for pattern in SECRET_PATTERNS:
        if pattern in key_lower:
            if isinstance(value, str) and len(value) > 4:
                return value[:2] + "***" + value[-2:]
            return "***"
    return value


def redact_dict(d):
    """Return a copy of a dict with secret values redacted."""
    return {k: redact_value(k, v) for k, v in d.items()}
```

```python
#!/usr/bin/env python3
"""Idempotent resource manager exercise."""

import json
import time
import logging
from pathlib import Path
from dataclasses import dataclass, field
from typing import Any

logger = logging.getLogger(__name__)


# --- Simulated Resource Store ---

class ResourceStore:
    """Simulates a resource backend (like a cloud API or database)."""

    def __init__(self, state_file):
        self.state_file = Path(state_file)
        self._resources = self._load()

    def _load(self):
        if self.state_file.exists():
            return json.loads(self.state_file.read_text())
        return {}

    def _save(self):
        self.state_file.parent.mkdir(parents=True, exist_ok=True)
        self.state_file.write_text(json.dumps(self._resources, indent=2))

    def exists(self, resource_type, name):
        return name in self._resources.get(resource_type, {})

    def get(self, resource_type, name):
        return self._resources.get(resource_type, {}).get(name)

    def list_resources(self, resource_type):
        return dict(self._resources.get(resource_type, {}))

    def create(self, resource_type, name, spec):
        if resource_type not in self._resources:
            self._resources[resource_type] = {}
        self._resources[resource_type][name] = {
            "spec": spec,
            "created_at": time.strftime("%Y-%m-%dT%H:%M:%SZ"),
            "updated_at": time.strftime("%Y-%m-%dT%H:%M:%SZ"),
        }
        self._save()

    def update(self, resource_type, name, spec):
        if self.exists(resource_type, name):
            self._resources[resource_type][name]["spec"] = spec
            self._resources[resource_type][name]["updated_at"] = time.strftime("%Y-%m-%dT%H:%M:%SZ")
            self._save()

    def delete(self, resource_type, name):
        if self.exists(resource_type, name):
            del self._resources[resource_type][name]
            self._save()


# --- Idempotent Resource Manager ---

@dataclass
class ReconcileResult:
    created: list = field(default_factory=list)
    updated: list = field(default_factory=list)
    deleted: list = field(default_factory=list)
    unchanged: list = field(default_factory=list)
    errors: list = field(default_factory=list)

    @property
    def changed(self):
        return bool(self.created or self.updated or self.deleted)

    def summary(self):
        return (f"created={len(self.created)}, updated={len(self.updated)}, "
                f"deleted={len(self.deleted)}, unchanged={len(self.unchanged)}, "
                f"errors={len(self.errors)}")


class ResourceManager:
    """Idempotent resource manager using desired-state reconciliation."""

    def __init__(self, store):
        self.store = store

    def reconcile(self, resource_type, desired_resources, delete_missing=False):
        """
        Reconcile current state with desired state.

        Args:
            resource_type: The type of resource (e.g., "servers", "configs")
            desired_resources: Dict of {name: spec} for desired state
            delete_missing: If True, delete resources not in desired state
        """
        result = ReconcileResult()
        current_resources = self.store.list_resources(resource_type)

        # Process desired resources
        for name, desired_spec in desired_resources.items():
            try:
                current = self.store.get(resource_type, name)

                if current is None:
                    # Resource does not exist — create it
                    logger.info(f"  CREATE {resource_type}/{name}")
                    self.store.create(resource_type, name, desired_spec)
                    result.created.append(name)

                elif current["spec"] != desired_spec:
                    # Resource exists but differs — update it
                    logger.info(f"  UPDATE {resource_type}/{name}")
                    logger.debug(f"    Current: {current['spec']}")
                    logger.debug(f"    Desired: {desired_spec}")
                    self.store.update(resource_type, name, desired_spec)
                    result.updated.append(name)

                else:
                    # Resource matches desired state — no action
                    logger.debug(f"  OK     {resource_type}/{name}")
                    result.unchanged.append(name)

            except Exception as e:
                logger.error(f"  ERROR  {resource_type}/{name}: {e}")
                result.errors.append({"name": name, "error": str(e)})

        # Delete resources not in desired state (if enabled)
        if delete_missing:
            for name in current_resources:
                if name not in desired_resources:
                    try:
                        logger.info(f"  DELETE {resource_type}/{name}")
                        self.store.delete(resource_type, name)
                        result.deleted.append(name)
                    except Exception as e:
                        logger.error(f"  ERROR deleting {resource_type}/{name}: {e}")
                        result.errors.append({"name": name, "error": str(e)})

        return result

    def ensure(self, resource_type, name, spec):
        """Ensure a single resource exists with the given spec (idempotent)."""
        current = self.store.get(resource_type, name)

        if current is None:
            self.store.create(resource_type, name, spec)
            return "created"
        elif current["spec"] != spec:
            self.store.update(resource_type, name, spec)
            return "updated"
        else:
            return "unchanged"

    def ensure_absent(self, resource_type, name):
        """Ensure a resource does NOT exist (idempotent)."""
        if self.store.exists(resource_type, name):
            self.store.delete(resource_type, name)
            return "deleted"
        return "already_absent"
```

```python
def execute_concurrent(func, items, config, correlation_id, **kwargs):
    """Execute a function across multiple items concurrently."""
    results = []
    max_workers = min(config.workers, len(items))

    logger.info(
        f"Executing on {len(items)} targets with {max_workers} workers",
        extra={"correlation_id": correlation_id},
    )

    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        futures = {
            executor.submit(func, item, config=config,
                          correlation_id=correlation_id, **kwargs): item
            for item in items
        }

        for future in as_completed(futures):
            item = futures[future]
            try:
                result = future.result()
                results.append(result)
            except Exception as e:
                results.append(OperationResult(
                    target=item, action="unknown",
                    success=False, message=str(e),
                ))

    return results

        print_results(results, f"Deploy {image}", cid)

        failed = [r for r in results if not r.success]
        sys.exit(1 if failed else 0)

    @cli.command("show-config")
    @click.pass_context
    def show_config(ctx):
        """Show current configuration (secrets redacted)."""
        config = ctx.obj["config"]
        print(json.dumps(config.to_safe_dict(), indent=2))

    @cli.command()
    def health(ctx=None):
        """Health check for this tool itself."""
        checks = {
            "python": sys.version.split()[0],
            "hostname": socket.gethostname(),
            "click": "available" if HAS_CLICK else "missing",
            "requests": "available" if HAS_REQUESTS else "missing",
        }
        print(json.dumps({"status": "ok", "checks": checks}, indent=2))

def main_fallback():
    """Simple entry point when Click is not installed."""
    config = load_config()
    setup_logging(level=config.log_level)

    cid = str(uuid.uuid4())[:8]
    servers = config.servers or ["web-01", "web-02", "db-01"]

    print("Running health check (install 'click' for full CLI)...\n")
    results = execute_concurrent(check_server, servers, config, cid)
    print_results(results, "Health Check", cid)


# ============================================================

def main():
    if HAS_CLICK:
        cli()
    else:

if __name__ == "__main__":
```

## Key Takeaways

- Configuration hierarchy: defaults < config file < environment variables < CLI arguments
- Use Pydantic or dataclass validation to fail fast on invalid configuration at startup
- Never put secrets in code or version control — load them from environment variables, `.env` files, or vault services
- Redact secrets in log output by checking key names for patterns like "password", "secret", "token"
- Idempotent operations are safe to run multiple times — use the check-before-act and desired-state patterns
- Use `ThreadPoolExecutor` for concurrent I/O operations (checking multiple servers, deploying in parallel)
- `asyncio` with `aiohttp` is better for very high concurrency (hundreds of concurrent connections)
- Retry transient failures with exponential backoff — the `tenacity` library handles this elegantly
- Structured logging (JSON) makes logs parseable by aggregation tools (ELK, Datadog, CloudWatch)
- Correlation IDs link related log entries across a multi-step operation
- Package tools with `pyproject.toml` and entry points for clean `pip install` and system-wide commands
- Run through the production checklist before any automation tool goes live
